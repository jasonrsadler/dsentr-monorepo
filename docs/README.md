# Dsentr Documentation

This directory contains all public documentation for Dsentr. Each section maps directly to a functional area of the product.

Contents
- Nodes: reference pages for triggers, actions, logic blocks, configuration fields, and execution behavior.
- Edges: details on how nodes connect, data propagation rules, error flow, and execution order.
- Settings: workspace settings, billing, usage, integrations, secrets, webhooks, and engine controls.
- User Profile: account-level settings, authentication, profile fields, and personal configuration.
- Workflow Templates: prebuilt workflows and instructions for customizing, importing, and publishing templates.
- Workflow Authentication: how workflows authenticate to third-party services, token handling, and permission scopes.
