# dsentr documentation site

A standalone Vite + React application that delivers customer-facing documentation for the dsentr automation platform.

## Getting started

```bash
npm install
npm run dev
```

Visit `http://localhost:4174` to view the docs site locally.

## Available scripts

- `npm run dev` – start the Vite dev server.
- `npm run build` – type-check and build the production bundle.
- `npm run preview` – preview the production build locally.
- `npm run lint` – run ESLint across the project.
- `npm run test` – run Vitest in watch mode.
- `npm run test:run` – run Vitest once, suitable for CI.
